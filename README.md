Задодить в репозиторий structures_practicum, нажать на зеленую кнопку Code и Download ZIP. После этого извлекаешь файл и заходишь в PyCharm или в Pyhton. Нажимаешь на 4 полоски и New Project... и в Location надо нажать папку, после этого находишь разархивированный файл который вы скачали, выберайте и нажмите на Create.
